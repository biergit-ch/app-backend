# Biergit app-backend

## Introduction

*Backend of the **biergit** app*

## Setup

1. Install the latest version of go
2. Configure `GOPATH` var
3. Install **go** packages with **go get**

## Commands

### Development Server

`go run main.go` builds the backend and listens on port 8080
